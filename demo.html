11 Feb 12
Incredible for someone who has little QA knowledge.

I started this book after a little warm up in Quants with the help of a different study material, so which implies that I knew some concepts already before turning the cover page of this book. The first day with this book was really enjoyable. The theory part is explained in an apt way along with the problem exercise. So one can expect thorough concept building plus problem practice from this book. I haven't finished the book but so far so good. The different levels of difficulty are providing useful information and I'm getting a good grasp of things out of this book. Also, I have been told by experts that this is not the only ladder to climb if some one is seriously preparing for CAT. But this book is certainly of great help if the person is not an expert already. Experts can ignore my suggestion. :D So if you are starting your preparation or some where in the middle of it, you must go through it. Happy Learning
